
STREAMLIT GEMINI HEALTH DIAGNOSTICS (FREE VERSION)

SETUP:

1. conda activate healthAI
2. pip install -r requirements.txt

IMPORTANT:
Install Tesseract OCR (for PNG support)
Download:
https://github.com/UB-Mannheim/tesseract/wiki

Add to PATH.

GET GEMINI FREE API KEY:
https://makersuite.google.com/app/apikey

Paste key in .env:
GEMINI_API_KEY=your_real_key_here

Run:
conda activate healthAI
cd Downloads
dir
cd Streamlit_Gemini_Health_Diagnostics
streamlit run app.py

Open:
http://localhost:8501
